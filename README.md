Jovis Collection

Jovis Collection is an elegant and modern e-commerce website showcasing a curated selection of clothing, accessories, and lifestyle products. Designed with a user-friendly interface, the website provides seamless navigation, beautiful product displays, and a responsive design for all devices.

Features:
	•	Product catalog with categories for easy browsing
	•	Responsive design optimized for desktop and mobile
	•	Simple and intuitive navigation
	•	High-quality images and product descriptions
	•	Contact and social media integration

Tech Stack:
	•	Frontend: HTML, CSS, JavaScript
	•	Backend  Python Flask /Djagon
	•	Database: SQLite / MySQL

Purpose:
This project is a portfolio-ready showcase for demonstrating web development skills, UI/UX design, and responsive e-commerce layouts.
